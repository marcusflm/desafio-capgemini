# Desafio Capgemini

Este projeto foi desenvolvido com a finalidade de participar do desafio de programação da Academia Capgemini.

## Iniciando

Para executar o projeto é necessário ter os seguintes programas instalados:

- [JDK 8 ou superior: Necessário para executar o projeto Java](http://www.oracle.com/technetwork/java/javase/downloads/jdk10-downloads-4416644.html)
- [Eclipse: Para desenvolvimento do projeto](http://www.eclipse.org/downloads/packages/eclipse-ide-java-ee-developers/oxygen3a)

## Executando

- Questão 01: executar o método *Main* da classe Escada e inserir um número inteiro;

- Questão 02: executar o método *verificaSenha* passando como parâmetro uma *String* para ser verificada. O método retornará o número de caracteres que deverão ser incluídos à *String* para que seja considerada uma **senha forte**. Se for o caso, a questão possui uma classe para teste unitário. 

- Questão 03: executar o método *verificaAnagramas* passando como parâmetro uma *String* para ser verificada. O método retornará o número de pares de **anagramas** que podem ser formados com a *String* inserida.
